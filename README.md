# Frontend Archive

프론트엔드 학습 과정 속에서 다양한 인터랙션, 애니메이션, 레이아웃, UI 패턴을 직접 구현하고 기록하는 아카이브입니다.

## 목표

- HTML/CSS/JavaScript 구현 능력 향상
- UI 컴포넌트와 인터랙션 패턴 반복 학습
- 디자인을 코드로 구현하는 감각 훈련
- 작은 결과물을 꾸준히 쌓아 포트폴리오 보조 자료로 활용

## 카테고리

1. UI Components
2. Layout Practice
3. Animation & Motion
4. CSS Drawing
5. Mini Games
6. 3D & Advanced
7. Clone & Rebuild

## 기록 방식

각 실습 폴더에는 다음 내용을 기록합니다.

- 구현 목표
- 사용 기술
- 주요 기능
- 어려웠던 점
- 개선할 점
